You need to sort the list of dictionaries in descending order of 'price' and leave as many dictionaries as defined via 'limit' parameter.
What exactly should I use for this?
Use the built-in sorted() function.
How exactly the sorting of dictionaries will be implemented according to the 'price' values?
The function might look like this: 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'])
But the dictionaries were sorted from the lowest to the highest 'price'. What should I do?
You can add an extra parameter: 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'], reverse=True)
How exactly should I take into account the 'limit' parameter?
In order to reduce the final sorted list to the number of elements equal to the 'limit', you should make a slice and specify the value of 'limit' as the right limit. It'll look something like this: 
x = sorted(x)[:limit]
