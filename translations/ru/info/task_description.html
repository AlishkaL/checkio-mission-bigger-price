Дана таблица всех доступных продуктов на складе. Данные представленны в виде {% if interpreter.slug == "js-node" %} массива объектов (array of objects){% else %}списка диктов (a list of dicts){% endif %}

Your mission here is to find TOP most expensive goods. The amount we are looking for will be given as a first argument and the whole data as the second one

Ваша миссия тут - это найти ТОП самых дорогих товаров. Количество товаров, которые мы ищем, будет переданно в первом аргументе, а сами данные по товарам будут переданны вторым аргументом.

{% if interpreter.slug == "js-node" %}

Input: Число и массив объектов (Integer and array of objects). У каждого объекта 2 аттрибута "name" и "price"

{% else %}

Input: Число и массив диктов (int and list of dicts). Каждый дикт имеет 2 ключа "name" и "price".

{% endif %}

Output: Такой же как и второй аргумент.

Example:

{% if interpreter.slug == "js-node" %} 
biggerPrice(2, [
    {"name": "bread", "price": 100},
    {"name": "wine", "price": 138},
    {"name": "meat", "price": 15},
    {"name": "water", "price": 1}
]) == [
    {"name": "wine", "price": 138},
    {"name": "bread", "price": 100}
]

biggerPrice(1, [
    {"name": "pen", "price": 5},
    {"name": "whiteboard", "price": 170}
]) == [{"name": "whiteboard", "price": 170}]
{% else %} 
bigger_price(2, [
    {"name": "bread", "price": 100},
    {"name": "wine", "price": 138},
    {"name": "meat", "price": 15},
    {"name": "water", "price": 1}
]) == [
    {"name": "wine", "price": 138},
    {"name": "bread", "price": 100}
]

bigger_price(1, [
    {"name": "pen", "price": 5},
    {"name": "whiteboard", "price": 170}
]) == [{"name": "whiteboard", "price": 170}]
{% endif %}
