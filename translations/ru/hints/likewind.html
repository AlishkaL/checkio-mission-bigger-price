Вам необходимо отсортировать список словарей по убыванию значения 'price' и оставить столько словарей, сколько указано в параметре 'limit'.
Что именно я должен использовать для этого?
Используйте встроенную функцию sorted().
Как именно будет реализовываться сортировка словарей по значениям 'price'?
Функция может выглядеть так: 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'])
Но словари отсортировались от меньшего 'price' к большему. Что делать?
Достаточно добавить дополнительный параметр: 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'], reverse=True)
Как именно я должен учесть параметр 'limit'?
Для того, чтобы ограничить итоговый отсортированный список количеством элементов, равным 'limit', достаточно сделать срез и в качестве правого предела указать значение 'limit'. Условно, это будет выглядеть так: 
x = sorted(x)[:limit]
Мне нужно увидеть полную картину, чтобы разобраться.
Готовая функция может выглядеть так: 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'], reverse=True)[:limit]
