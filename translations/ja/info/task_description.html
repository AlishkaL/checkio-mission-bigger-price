店に売られているすべての品物のテーブルがあります。このデータは{% if interpreter.slug == "js-node" %}オブジェクト{% else %}辞書{% endif %}のリストで表されます。

このミッションでは、最も高額な品物を見つけてください。見つける数は最初の引数で与えられ、データ全体が二番目の引数で与えられます。

{% if interpreter.slug == "js-node" %}

入力: 整数とobjectのarray。各objectは "name" と "price" の属性をもっている。

{% else %}

入力: 整数と辞書のリスト。 各辞書は "name" と "price" のキーをもっている。

{% endif %}

出力: 入力の二番目と同じ。

例:

{% if interpreter.slug == "js-node" %} 
biggerPrice(2, [
    {"name": "bread", "price": 100},
    {"name": "wine", "price": 138},
    {"name": "meat", "price": 15},
    {"name": "water", "price": 1}
]) == [
    {"name": "wine", "price": 138},
    {"name": "bread", "price": 100}
]

biggerPrice(1, [
    {"name": "pen", "price": 5},
    {"name": "whiteboard", "price": 170}
]) == [{"name": "whiteboard", "price": 170}]
{% else %} 
bigger_price(2, [
    {"name": "bread", "price": 100},
    {"name": "wine", "price": 138},
    {"name": "meat", "price": 15},
    {"name": "water", "price": 1}
]) == [
    {"name": "wine", "price": 138},
    {"name": "bread", "price": 100}
]

bigger_price(1, [
    {"name": "pen", "price": 5},
    {"name": "whiteboard", "price": 170}
]) == [{"name": "whiteboard", "price": 170}]
{% endif %}
