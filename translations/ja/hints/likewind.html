最初に実装のアイデアが必要です。
まず辞書のリストを'price'の降順にソートすればよいでしょう。 そして、その結果を'limit'の数だけ残しましょう。
では、何を使えばいいのでしょう？
組み込みの sorted()関数が便利です。
どうやって辞書を'price'の値の順にソートすればいいのですか？
このような関数です:（訳者注：sorted()にはkey関数を与えることができ、この順に並べ替えられます） 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'])
'price'の小さい順にソートされてしまいました。どうすればいいですか？
新しくパラメータを一つ付け加えます: 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'], reverse=True)
どうやって'limit'のパラメータを考慮すればいいのでしょう。
ソート結果を'limit'の数だけに制限するには、スライスを使ってちょうど'limit'の数だけになるように結果を切り出します。 このようになるでしょう: 
x = sorted(x)[:limit]

def bigger_price(limit, data):
