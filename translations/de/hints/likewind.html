Ich brauche ein paar Ideen für den Anfang.
Sie müssen die Liste der Wörterbücher in absteigender Reihenfolge nach 'Preis' sortieren und so viele Wörterbücher übrig lassen wie über den Parameter 'Limit' definiert.
Was genau soll ich dafür verwenden?
Verwenden Sie die eingebaute sorted() funktion.
Wie genau wird die Sortierung der Wörterbücher nach den 'Preis'-Werten umgesetzt?
Die Funktion könnte wie folgt aussehen: 
def bigger_price(limit: int, data: list[dict]) -> list[dict]:
    return sorted(data, key=lambda x: x['price'])
Aber die Wörterbücher wurden vom niedrigsten zum höchsten 'Preis' sortiert. Was soll ich tun?
Sie können einen zusätzlichen Parameter hinzufügen: 
def bigger_price(limit: int, data: list[dict]) -> list[dict]:
    return sorted(data, key=lambda x: x['price'], reverse=True)
Wie genau soll ich den Parameter 'Limit' berücksichtigen?
Um die endgültige sortierte Liste auf die Anzahl der Elemente zu reduzieren, die dem 'limit' entspricht, sollten Sie ein Slice erstellen und den Wert von 'limit' als rechte Grenze angeben. Das sieht dann etwa so aus: 
x = sorted(x)[:limit]
Um das herauszufinden, muss ich das ganze Bild sehen.
Die Funktion könnte wie folgt aussehen: 
def bigger_price(limit: int, data: list[dict]) -> list[dict]:
    return sorted(data, key=lambda x: x['price'], reverse=True)[:limit]
