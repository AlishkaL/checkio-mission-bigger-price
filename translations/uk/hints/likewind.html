Для початку мені потрібні ідеї.
Тобі потрібно сортувати список словників у порядок 'price' і залишити стільки словників як визначено за допомогою параметра 'limit'.
Що саме я маю використовувати для цього?
Використовуй вбудовану sorted() функцію.
Як саме сортування словників буде реалізовано відповідно до цінностей "ціни"?
Функція може виглядати так: 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'])
Але словники були відсортовані від найнижчої до найвищої "price". Що я маю зробити?
Ти можеш додати параметр "reverse": 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'], reverse=True)
Як саме я маю враховувати параметр "limit"?
Для того, щоб зменшити остаточний відсортований список до кількості елементів, що дорівнюють "limit", та повинен зробити зріз і вказати значення "limit" як правильної межі. Це буде виглядати приблизно так: 
x = sorted(x)[:limit]
Щоб зрозуміти це, мені потрібно побачити всю картину.
Функція може виглядати так: 
def bigger_price(limit, data):
    return sorted(data, key=lambda x: x['price'], reverse=True)[:limit]
